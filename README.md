# Portfolio — Imtius Ahmad

**Live:** [portfolio-two-kohl-35.vercel.app](https://portfolio-two-kohl-35.vercel.app)

---

## Built With

- **Next.js** — React framework with App Router and server components
- **TypeScript** — Type-safe development
- **Tailwind CSS** — Utility-first styling
- **Prisma** — ORM for PostgreSQL
- **PostgreSQL** — Database via Prisma Postgres
- **Vercel** — Hosting and serverless functions

---

## Features

- Responsive design across all devices
- Server-side rendering with dynamic routes
- Protected admin dashboard
- Contact form with database storage
- Project showcase with detail pages
- Typing animation and scroll effects

---

## Getting Started

```bash
git clone https://github.com/Imtius10/portfolio.git
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
